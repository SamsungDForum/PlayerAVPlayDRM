Tizen Video DRM App

Video player based on Tizen webapis.avplay

Available buttons/actions:

1 - Toggle UHD content playback - use before starting playback and only for 4K content
2 - display track info - while playback active
3 - display stream properties - while playback active

    Logs

    Info

    This application demonstrates the use of AVPlay based player with DRM

    The application downloads video content from http://playready.directtaps.net and http://commondatastorage.googleapis.com - it will not work if these domains are not accessible
    You can change the source for video content by modifying contents of drms variable in main.js file

    Operation

    Use mouse or remote to operate on video player.
    Use UP/DOWN arrows to select DRM system
    Available keys: PLAY, PAUSE, FW, RW, STOP, ENTER (Fullscreen), RETURN, PLAY/PAUSE (Smart Remote 2015)
